Claude Code

CLI dan extension resmi Anthropic untuk coding dengan AI di terminal atau editor.

  1. 1 Install Claude Code

    Buka terminal (PowerShell di Windows, Terminal di Mac/Linux), lalu jalankan perintah berikut:

    npm install -g @anthropic-ai/claude-code@latest

  2. 2 Buka folder Home

    Lokasi folder Home: Windows C:\Users\NamaAnda · Mac/Linux /home/namaanda

  3. 3 Buat folder .claude

    Di dalam folder Home, buat folder bernama .claude jika belum ada.

  4. 4 Konfigurasi settings.json

    Buat atau edit file .claude/settings.json. Ganti nilai API Key dengan milik Anda:

    {
  "env": {
    "ANTHROPIC_BASE_URL": "https://max-omni.mind-aku.my.id/v1",
    "ANTHROPIC_AUTH_TOKEN": "MASUKKAN-API-KEY-ANDA",
    "ANTHROPIC_MODEL": "mind/claude-opus-4.8",
    "ANTHROPIC_SMALL_FAST_MODEL": "mind/claude-opus-4.8"
  }
}

  5. 5 Jalankan Claude Code

    Setelah settings.json disimpan, buka terminal dan ketik:

    claude

  6. 6 Logout sesi lama

    Di dalam sesi Claude Code, ketik perintah berikut lalu tekan Enter:

    /logout

  7. 7 Buka ulang Claude Code

    Tutup terminal, buka terminal baru, lalu ketik lagi:

    claude

    Saat muncul pertanyaan konfirmasi, pilih jawaban pertama atau ketik yes untuk semua pilihan.

  8. 8 Uji koneksi

    Ketik prompt berikut:

    halo

    Jika AI menjawab, berarti konfigurasi sudah berhasil. Jika terjadi error, periksa kembali API Key dan Base URL.

OpenAI Codex Desktop

Aplikasi desktop resmi OpenAI untuk coding dengan AI.

Kuota mengikuti paket GPT Plus Codex pada akun private Anda (bukan akun sharing).

  1. 1 Tutup Codex sepenuhnya

    Di Windows, pastikan proses Codex juga ditutup dari Task Manager agar konfigurasi baru terbaca.

  2. 2 Edit config.toml

    Lokasi file: Windows C:\Users\Nama_User\.codex\config.toml · Mac: jalankan open ~/.codex/config.toml di terminal.

  3. 3 Tambahkan Base URL

    Tambahkan baris berikut di bagian paling atas file:

    openai_base_url = "https://max-omni.mind-aku.my.id/v1"

  4. 4 Logout dan buka ulang

    Buka Codex, logout jika sebelumnya sudah login, lalu lanjut ke langkah berikutnya.

  5. 5 Masuk dengan API Key

    Pilih Sign in another way, lalu masukkan API Key Anda.

  6. 6 Verifikasi dan pilih model

    Saat chat pertama kali, status "reconnecting" adalah normal — tunggu hingga respons muncul. Pilih model 5.5 high (satu-satunya model yang didukung).

OpenClaw

AI agent berbasis terminal untuk otomasi dan asisten coding.

  1. 1 Install OpenClaw

    Jika belum terinstall, unduh dan ikuti panduan di openclaw.ai.

  2. 2 Update ke versi terbaru

    openclaw update

  3. 3 Jalankan setup provider

    openclaw onboard --install-daemon

  4. 4 Isi wizard sesuai panduan

    • Model/auth provider → Custom Provider
    • API Base URL → https://max-omni.mind-aku.my.id/v1
    • API Key → API Key Anda
    • Endpoint compatibility → OpenAI-compatible
    • Model ID → claude-opus-4.8
    • Endpoint ID → omni

OpenCode Desktop

Aplikasi desktop OpenCode untuk coding dengan AI — hubungkan lewat Custom Provider di pengaturan.

  1. 1 Unduh dan install OpenCode Desktop

    Jika belum terinstall, unduh dari opencode.ai lalu ikuti petunjuk instalasi untuk Windows, Mac, atau Linux.

  2. 2 Buka aplikasi OpenCode

    Jalankan OpenCode Desktop setelah instalasi selesai.

  3. 3 Buka Settings

    Klik ikon gerigi (Settings) di aplikasi.

  4. 4 Hubungkan Custom Provider

    Buka menu Providers, gulir ke bawah, cari Custom Provider, lalu klik Connect.

  5. 5 Isi koneksi API

    • API Base URL → https://max-omni.mind-aku.my.id/v1
    • API Key → API Key Anda
    • Endpoint compatibility → OpenAI-compatible
    • Endpoint ID / Provider ID → omni

  6. 6 Isi Model ID

    Sesuaikan dengan paket API Key Anda:

    • Paket Opus → claude-opus-4.8
    • Paket GPT → gpt-5.5

  7. 7 Submit

    Klik Submit untuk menyimpan provider.

  8. 8 Buka project Anda

    Buka folder project yang ingin Anda kerjakan di OpenCode.

  9. 9 Pilih model di chat

    Di panel chat, pilih model omni (atau nama provider yang Anda set di langkah 5), lalu uji dengan pesan singkat.
